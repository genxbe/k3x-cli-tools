## Installation

### Download

Download and copy this repository to `/site/plugins/k3x-cli-tools`.

### Git submodule

```
git submodule add https://github.com/genxbe/k3x-cli-tools.git site/plugins/k3x-cli-tools
```

### Composer

```
composer require genxbe/k3x-cli-tools
```
